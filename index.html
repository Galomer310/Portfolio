Pub & Restaurant Expense Calculator

Overview : This web app manages user interactions and calculations for the Pub & Restaurant Expense Calculator. It captures user input regarding the costs of drinks and food, calculates the total expenses.

Key Features: Event Listeners : Each interactive element (beer, shot, food, tip) has an event listener that triggers when clicked, prompting the user for relevant input.
Calculations : When the beer button is clicked: It prompts for the cost of one beer and the number of beers consumed. Calculates the total beer payment and updates the corresponding display. The same logic applies to the shot and food buttons, with specific prompts tailored to each item. For the food cost, it divides the total by the number of participants to ensure each person’s share is calculated accurately.
Tip Calculation : It prompts for the desired tip percentage. Calculates the tip based on the total count of expenses and adds it to the total amount due.
Enhancing Functionality with JavaScript : Enhanced understanding of JavaScript basics, including variable scope and data types. Gained experience with event handling and DOM manipulation to create an interactive user experience.
Error Handling and User Experience : Realized the importance of validating user input and handling potential errors (e.g., ensuring valid numbers are entered) to prevent incorrect calculations.
